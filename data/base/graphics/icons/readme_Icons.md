# Icons are sorted into different folders:
## unused
Contains unused Icons - maybe for later
## unchanged 
Contains Icons extracted from the main game and are used by the mod. 
These are here in case Ubisoft decides to remove/change any icons in a future patch.
## other
Contains Icons used by the mod and are changed variations of Icons from Ubisoft. 
Changed .kra file (Krita) should be in the repo for further modifications. 
### animals
Contains Animal Icons. Currently mostly from 1800 and modified with filters. 
## <name> e.g. celtic_druid
Contains a template file and varitaions of an item. For example an base Face of a celtic druid but with different clothes based on followed deity
## helpertemplates
Contains templates for various Items to be used like Bow, Arrows, Amphore, etc... 

## Naming Convention
Icons should use similar naming as the Item they are used by. 
This should help to link them with the corresponding Item.
Also the name should contain the gender of the Icon. This helps later for selecting corresponding Barks (AudioOutput when socketing the item).
